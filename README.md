🎧 Spotify Music Insights Dashboard

📊 Overview

This project explores Spotify music data to uncover patterns in song popularity and audio features. Using Tableau, an interactive dashboard was built to analyze trends in energy, danceability, duration, and release patterns, providing data-driven insights into what makes songs successful.

🚀 Key Features
- 🎵 Top 10 Songs by Popularity – Identifies highest-performing tracks
- 📈 Tracks Released Over Time – Shows growth of music releases across years
- ⚡ Energy vs Danceability – Explores relationship between key audio features
- 🔞 Explicit Content Analysis – Compares popularity of explicit vs non-explicit songs
- ⏱️ Song Duration Distribution – Highlights common track length patterns

📌 Insights
- Popular songs tend to have balanced energy and high danceability
- Significant growth in music releases observed over recent decades
- Most songs fall within a standard duration range (approx. 3–5 minutes)
- Audio features play a crucial role in defining listener preferences

🛠️ Tools & Technologies
- Tableau Public – Data visualization & dashboard creation
- Dataset – Spotify tracks dataset (audio features & metadata)
